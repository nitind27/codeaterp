🚀 Flutter API URLs

CodeatERP - All API Endpoints for Flutter App

⚙️ Base URL Configuration

🔐 Authentication APIs

POST
Login - Use skipLocationCheck: true to bypass location
GET Auth Required
Get current logged in user details
POST Admin/HR
Register new user (Admin/HR only)

📊 Dashboard API

GET Auth Required
Get complete dashboard data (attendance, tasks, leaves, etc.)

⏰ Attendance APIs

GET Auth Required
Get attendance records (supports ?start_date=&end_date=&page=&limit=)
GET Auth Required
Get today's attendance status
POST
Clock In
PUT
Clock Out
GET
Get single attendance record

👥 Employee APIs

GET Admin/HR/PM
Get all employees (supports ?search=&department=&status=&page=&limit=)
POST Admin/HR
Create new employee
GET
Get single employee details
PUT
Update employee
DELETE Admin/HR
Delete employee

🏖️ Leave APIs

GET
Get leave applications (supports ?status=&start_date=&end_date=)
POST
Apply for leave
PUT Admin/HR
Approve/Reject leave
GET
Get leave balance
GET
Get all leave types

📋 Project APIs

POST Admin/HR/PM

GET
